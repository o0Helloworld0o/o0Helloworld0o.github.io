第7单元

发表于 2019-03-24 | 分类于 新标日

练习I
1.
[例1] 李さんにあげます／本 → これは李さんにあげる本です。
（1）課長に見せます／手紙 → これは課長に見せる手紙です。
（2）図書館で借りました／雑誌 → これは図書館で借りた雑誌です。（注意过去时）
（3）1日に3回飲みます／薬 → これは1日に3回飲む薬です。
（4）甘くて美味しいです／飲み物 → これは甘くて美味しい飲み物です。（上册就有了）
（5）日本にはありません／果物 → これは日本にはない果物です。（应该使用は还是が？）
（6）綺麗で優しかったです／母の写真 → これは綺麗で優しかった母の写真です。

[例2] 子供／読みます／絵本 → これは子供が読む絵本です。
（7）森さん／カラオケでよく歌います／中国の歌 → これは森さんがカラオケでよく歌う中国の歌です。
（8）私／生まれました／家の写真 → これは私が生まれた家の写真です。
（9）陳さん／まだ食べたことがありません／日本料理 → これは陳さんがまだ食べたことのない日本料理です。（参考答案使用の）
（10）李さん／明日の会議で使います／資料 → これは李さんが明日の会議で使う資料です。

[例3] 昨日会社を休みました → 昨日会社を休んだ人は森さんです。
（11）明日会社へ来ません → 明日会社へ来ない人は森さんです。
（12）山でけがをしました → 山でけがをした人は森さんです。
（13）まだ結婚していません → まだ結婚していない人は森さんです。（います属于二类动词）
（14）周さんを知りませんでした → 周さんを知らなかった人は森さんです。

2.
[例] テレビを見ている人はだれですか。
（1）手紙を書いている人はだれですか。
（2）お茶を飲んでいる人はだれですか。
（3）タバコを吸っている人はだれですか。
（4）新聞を読んでいる人はだれですか。
（5）電話をかけている人はだれですか。

3.
[例1] 中国で買いました／CDを友達に貸しました → 中国で買ったCDを友達に貸しました。
（1）李さんが書きました／レポートを読みました → 李さんが書いたレポートを読みました。
（2）日本で撮りました／ビデオを見ませんか → 日本で撮ったビデオを見ませんか。
（3）明日泊まります／ホテルの電話番号を教えてください → 明日泊まるホテルの電話番号を教えてください。
（4）父にもらいました／時計をなくしました → 父にもらった時計をなくしました。

[例2] 操作が簡単です／パソコンが欲しいです → 操作が簡単なパソコンが欲しいです。
（5）デザインが新しいです／靴を買いたいです → デザインが新しい靴を買いたいです。
（6）自然が豊かです／国が少なくなりました → 自然が豊かな国が少なくなりました。
（7）あなたの子供が好きです／料理は何ですか → あなたの子供が好きな料理は何ですか。
（8）仕事が面白くて，給料が高いです／会社に入りたいです → 仕事が面白くて，給料が高い会社に入りたいです。

4.
[例1] 小野さんにもらいました／CDを聞きます → 小野さんにもらったCDを聞いていますが。
（1）日本から来ました／メールをチェックします → 日本から来たメールをチェックしていますが。
（2）先月の旅行で撮りました／写真を見ます → 先月の旅行で撮った写真を見ていますが。
（3）日本の友達に出します／手紙を書きます → 日本の友達に出す手紙を書いていますが。（出します是一类动词）

[例2] 今，田中さんと話しています／見ました → 今，田中さんと話している人です。見たことはあるんですが，名前はちょっと…。
（4）あそこでコピーを取っています／話しました → あそこでコピーを取っている人です。話したことはあるんですが，名前はちょっと…。
（5）今，李さんとお茶を飲んでいます／顔を見ました → 今，李さんとお茶を飲んでいる人です。顔を見たことはあるんですが，名前はちょっと…。
（6）さっき入り口で会いました／一度会いました → さっき入り口で会った人です。一度会ったことはあるんですが，名前はちょっと…。
（7）あそこで電話をかけています／一度話しました → あそこで電話をかけている人です。一度話したことはあるんですが，名前はちょっと…。
（8）窓のところにいます／何度か顔を見ました → 窓のところにいる人です。何度か顔を見たことはあるんですが，名前はちょっと…。

练习II
1.
[例] 去年JC企画に入った社員は何人ですか。――――20人です。
（1）エレベーターの前にいる人を知っていますか。――――いいえ。だれでしょうね。
（2）あなたが買いたいかばんはどれですか。――――あの小さくて赤いのです。
（3）フランス語ができる人はだれですか。――――田中さんだと思います。
（4）机の上にあるパソコンを使ってもいいですか。――――すみません，ちょっと…。

2.
[例] 今パソコンを使っている人はだれですか。――――李さんで。
（1）小野さんが中国へ行く日はいつですか。――――あさってです。
（2）それはどこで撮った写真ですか。――――北京です。
（3）昨日聞いたCDはどうでしたか。――――良かったですよ。
（4）あなたが生まれた町はどん所ですか。――――静かな所です。
（5）今年JC企画に入った人の名前は何と言いますか。――――山田さんです。
